# Fazt API

## Requirements

- Docker (Optional)
- Postgres
- Node
- NPM

## First Steps

### Install Dependencies

```bash
npm install
```

## Development

### Run Development Server

```bash
docker-compose start # if you have docker
npm run dev
```

### Create Production Build

```
npm run build
```
